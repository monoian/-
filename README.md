# ✈️ 旅遊預算小工具 / Travel Budget Tool

一個輕量、純前端的旅遊預算規劃工具。輸入你的旅行天數與各項花費，即可立即算出**總預算**、**每日平均花費**與**最大支出項目**。

完全使用 **HTML + CSS + JavaScript** 製作，**不需要** npm、React、Next.js、後端、資料庫或任何 API key，打開瀏覽器即可使用。

---

## 📋 功能介紹

- **多項目預算輸入**：旅行天數、機票、住宿、當地交通、餐飲、景點門票、購物與其他。
- **一鍵計算**：點擊「計算」後立即顯示：
  - 💰 **總預算**：所有項目加總。
  - 📅 **每日平均花費**：總預算 ÷ 旅行天數。
  - 👑 **最大支出項目**：花費最高的類別。
  - 📊 **支出明細**：各項目金額一覽，最大項目特別標示。
- **清除按鈕**：一鍵清空所有欄位與結果，重新規劃。
- **輸入驗證**：天數需至少 1 天、需至少輸入一項金額，否則會提示。
- **全繁體中文介面**。
- **響應式設計（RWD）**：手機、平板、桌機皆有良好顯示。

## 🚀 使用方式

### 方法一：直接開啟
1. 下載或 clone 本專案。
2. 直接用瀏覽器打開 `index.html` 即可使用。

```bash
git clone https://github.com/monoian/travel-budget-tool.git
cd travel-budget-tool
# 用瀏覽器開啟 index.html
```

### 方法二：操作步驟
1. 輸入**旅行天數**。
2. 填入各項目的預算金額（沒有的項目可留空，視為 0）。
3. 點擊 **「計算」**，下方即顯示結果與明細。
4. 想重新規劃時，點擊 **「清除」** 清空所有資料。

## 🌐 發佈到 GitHub Pages

讓任何人都能透過網址直接使用本工具：

1. 將專案推送到 GitHub repository。
2. 進入 repository 頁面，點選上方 **Settings**（設定）。
3. 在左側選單找到 **Pages**。
4. 在 **Build and deployment → Source** 選擇 **Deploy from a branch**。
5. **Branch** 選擇 `main`（或你的主分支），資料夾選擇 `/ (root)`，按 **Save**。
6. 等待約 1 分鐘，GitHub 會產生網址，格式為：

   ```
   https://<你的帳號>.github.io/travel-budget-tool/
   ```

7. 打開該網址即可線上使用 🎉

> 提示：若頁面未即時更新，請稍候片刻或清除瀏覽器快取後重新整理。
